# STM32F405-INAV

# FC Specifications:

* MCU: 168MHz 
* IMU: MPU6500 OR BMI160
* OSD
* 6x Uarts
* 6x PWM outputs (2x Motors, 4x Servos)
* 1x I2C
* 4x ADC (VBAT, Current, RSSI, AirSpeed)
* 1x Camera Input
* PPM receiver: not supported
* IBUS and CRSF supported
* No Built in inverter for SBUS input (buy or DIY a converter)
